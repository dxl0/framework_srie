SRIE > TP2 - Scanning Networks > Ping an IP address

The network scanning function allows you to utilize the Nmap command to survey the connected machines within a network and retrieve their corresponding IP addresses. This enables you to gain insights into the network topology and identify active hosts.

Command Used

The "nmap" tool is used in the command line shell. We will use two options:

{% raw %}$ nmap -T3 -sn {IP} {% endraw %}

Command Execution

Please put "*" on the octet you want to scan. For example, 192.168.28.* will scan ip adress from 192.168.28.0 to .255
