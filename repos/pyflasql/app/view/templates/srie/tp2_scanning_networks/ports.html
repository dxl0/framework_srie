SRIE > TP2 - Scanning Networks > See ports of a computer

Nmap command is an opensource tool that scans the network. It is able to list the different services of a computer connected to your network. In this page nmap will be used in order to check the different ports of a specified IP.

Command Used

The "nmap" tool is used in the command line shell. We will use one option:

{% raw %}$ nmap -sS {IP} {% endraw %}
