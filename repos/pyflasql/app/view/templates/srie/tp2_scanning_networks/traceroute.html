TP2 - Scanning Networks > Trace a route

Traceroute is an invaluable utility for mapping the path of data packets from their source to a specified destination. By leveraging this tool, you can trace the network route taken by the data, identifying each intermediate hop and gauging the latency and potential network bottlenecks along the way. This aids in troubleshooting network connectivity issues and optimizing network performance.

Command Used

The traceroute tool can be used in the shell:

{% raw %}$ traceroute {IP} {% endraw %}
