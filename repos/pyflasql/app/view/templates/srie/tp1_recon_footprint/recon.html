TP1 - Reconnaissance / Footprint > Recon-ng

Recon-ng is a web-based open-source reconnaissance tool (OSINT) written in Python, often paired with the Kali Linux penetration distribution. The tool reduces time spent harvesting information from open resources and consists of an extensive range of modules and database interaction.

Installation

The tool can be installed through the command line :

{% raw %}sudo apt install recon-ng{% endraw %}

Run

You can only run Recon-ng from the terminal

{% raw %}recon-ng{% endraw %}

Quick start

The tool can be installed through the command line :

{% raw %}sudo apt install recon-ng{% endraw %}


From the console, it is easy to get help by typing help after launching the framework.


Recon-ng begins with an empty framework. You will need to install modules and packages that are useful for you.

Create a workspace

You will first need to create a workspace by typing :

{% raw %}workspaces create name_of_the_workspace{% endraw %}


The following command will opens directly to that workspace

{% raw %}recon-ng -w name_of_the_workspace{% endraw %}

Modules

You can get help and a list of all the modules by typing :

{% raw %}marketplace help{% endraw %} {% raw %}marketplace search{% endraw %}


You can install modules by typing :

{% raw %}marketplace install module_name{% endraw %}


Then, you will need to load the module :

{% raw %}modules load module_name{% endraw %}


Finally, you can run the module :

{% raw %}run{% endraw %}
