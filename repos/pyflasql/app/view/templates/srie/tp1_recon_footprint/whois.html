TP1 - Reconnaissance / Footprint > WHOIS

This tool enables the user to get information about a domain.

Command Used

The "whois" tool is used in the command line shell. So far it takes one parameter:

{% raw %}$ curl -s https://raphaelviera.fr/ismin/toolbox/whois/whois.php?domain={domain} {% endraw %}
{% raw %}$ whois {domain} {% endraw %}

Help

Please refer to the following information for help on how to use this tool:

  • {domain}

    The domain to be searched for

Command example:
{% raw %}$ curl -s https://raphaelviera.fr/ismin/toolbox/whois/whois.php?domain=raphaelviera.fr {% endraw %}
