SRIE > TP2 - Scanning Networks > Ping an IP address

Nmap command is an opensource tool that scans the network. It is able to list the different services of a computer connected to your network. In this page, nmap will be used in order to enumerate the differents computers connected on the network, based on an IP.

Command Used

The "nmap" tool is used in the command line shell. We will use two options:

{% raw %}$ nmap -T3 -sn {IP} {% endraw %}

Command Execution

Please put "*" on the octet you want to scan. For example, 192.168.28.* will scan ip adress from 192.168.28.0 to .255
