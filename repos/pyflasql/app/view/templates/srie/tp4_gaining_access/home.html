SRIE > TP4 - Gaining Access

Once you have done all the previous steps, you will need to gain access through the remote system. This step will introduce you to several methods to do so.

Useful tools used in this phase

Below is a list of tools that can be used for this pentesting phase:

  1. Hydra

    Hydra is a login cracker which supports numerous protocols to attack. It is easy to use since new modules are easy to add. This tool can also be used to show you how easy it is to gain access in your own machine.

  2. Exploiting VSFTPD backdoor manually
  3. Metasploit framework

    Matasploit is a framework that lots of cybercriminals use. As it is an open-source one, it can be easily customized to response your own need.
